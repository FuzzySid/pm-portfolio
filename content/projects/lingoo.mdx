## Overview

Lingoo is a trilingual dictionary app built on one insight: Hindi speakers already have a mental model for grammatical gender — the same system Spanish uses but English doesn't. Instead of learning Spanish through English, Lingoo lets Hindi speakers triangulate directly, surfacing grammar comparisons that mainstream apps ignore.

## Features

- **Instant Trilingual Lookup** — Single input returns definitions in English, Spanish, and Hindi
- **Hindi Nuance Engine** — Comparative grammar notes on gender alignment/divergence between Hindi and Spanish
- **Native Pronunciation** — Web Speech Synthesis API with language-specific accents
- **Persistent Word Library** — localStorage-backed personal vocabulary with cache-first retrieval
- **Smart Search History** — Deduplicated, timestamped lookup history with vocabulary stats
